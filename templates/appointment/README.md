# Appointment Template

Veteriner, klinik, doktor, psikolog, diyetisyen ve benzeri randevu odaklı işletmeler için hazırlanacak şablondur.

## Görsel kalite hedefi

Arrivio'daki gibi ferah, büyük hero alanlı, sağ tarafta panel/mockup kartı olan, güven veren ve mobilde güçlü görünen modern bir yapı kullanılacak.

## Ana bölümler

- Hero
- Hizmetler
- Uzmanlar
- Randevu formu
- Acil WhatsApp alanı
- Galeri
- Google Maps
- İletişim

## Admin ihtiyacı

- Randevu talepleri
- Status yönetimi
- WhatsApp geçişi
- Not ekleme
- Hizmet yönetimi
- Uzman/veteriner kartı yönetimi
