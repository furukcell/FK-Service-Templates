# İlk Müşteri Onboarding Checklist

Bu checklist ilk gerçek müşteri alınırken kullanılacaktır.

## 1. Ön görüşme

- [ ] Müşteri sektörü netleştirildi.
- [ ] Uygun şablon seçildi.
- [ ] Kurulum fiyatı söylendi.
- [ ] Ekstra istekler ayrıca fiyatlandırılacak denildi.
- [ ] Aylık abonelik olmadığı açıklandı.
- [ ] Domain/Firebase/hosting hesabının müşteri adına olacağı açıklandı.

## 2. Bilgi toplama

- [ ] `docs/customer-checklist.md` müşteriye göre dolduruldu.
- [ ] Logo alındı.
- [ ] Telefon/WhatsApp alındı.
- [ ] Adres ve harita linki alındı.
- [ ] Hizmet/ilan/randevu alanları alındı.
- [ ] Görseller alındı.
- [ ] Admin panel e-postası alındı.

## 3. Demo hazırlığı

- [ ] Müşteri config dosyası oluşturuldu.
- [ ] Renkler/logolar işlendi.
- [ ] Hizmet listesi işlendi.
- [ ] Personel/danışman kartları işlendi.
- [ ] Demo link oluşturuldu.
- [ ] Müşteriye ilk demo gönderildi.

## 4. Revizyon

- [ ] Yazım hataları düzeltildi.
- [ ] Telefon/adres kontrol edildi.
- [ ] WhatsApp linki kontrol edildi.
- [ ] Görseller kontrol edildi.
- [ ] Müşteri onayı alındı.

## 5. Firebase / hosting

- [ ] Firebase projesi müşteri adına açıldı.
- [ ] Firestore açıldı.
- [ ] Authentication Email/Password açıldı.
- [ ] Admin kullanıcı oluşturuldu.
- [ ] Storage açıldı.
- [ ] Firestore rules deploy edildi.
- [ ] Storage rules deploy edildi.
- [ ] Netlify/Vercel deploy yapıldı.
- [ ] Environment variables girildi.

## 6. Canlı test

- [ ] Web site mobilde açıldı.
- [ ] Form gönderildi.
- [ ] Firestore kaydı oluştu.
- [ ] Admin panelde kayıt göründü.
- [ ] Status değiştirildi.
- [ ] Admin notu kaydedildi.
- [ ] WhatsApp linki çalıştı.
- [ ] Emlak ise ilan ekleme test edildi.

## 7. Teslim

- [ ] Canlı domain bağlandı.
- [ ] Admin panel linki verildi.
- [ ] Admin giriş bilgileri müşteriye verildi.
- [ ] Kısa kullanım anlatımı yapıldı.
- [ ] Ödeme alındı.
- [ ] Ekstra işlerin ücretli olduğu tekrar belirtildi.

## 8. Son mesaj şablonu

```text
Merhaba, siteniz yayına alınmıştır.

Web site: ...
Admin panel: ...
Admin e-posta: ...

Randevu/talep/ilan kayıtları admin panelde listelenir. WhatsApp butonuyla müşteriye hızlı dönüş yapabilirsiniz.

Ekstra sayfa, özel tasarım değişikliği veya yeni özellik talepleri ayrıca fiyatlandırılır.
```
