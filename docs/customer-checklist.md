# Müşteriden Alınacak Bilgiler Checklist

Bu checklist, yeni müşteri sitesi kurulmadan önce doldurulmalıdır.

## 1. Genel bilgiler

- [ ] Firma adı
- [ ] Sektör
- [ ] Seçilen şablon: appointment / salon / real-estate
- [ ] Yetkili kişi adı
- [ ] Yetkili telefon
- [ ] Yetkili e-posta

## 2. Marka bilgileri

- [ ] Logo
- [ ] Ana renk
- [ ] Yardımcı renk
- [ ] Varsa slogan
- [ ] Varsa marka fotoğrafları

## 3. İletişim bilgileri

- [ ] Telefon
- [ ] WhatsApp numarası
- [ ] Adres
- [ ] Google Maps linki
- [ ] Instagram linki
- [ ] Facebook linki
- [ ] Web domain bilgisi

## 4. Hizmetler

- [ ] Hizmet adı
- [ ] Hizmet açıklaması
- [ ] Fiyat / başlangıç fiyatı
- [ ] Hizmet süresi
- [ ] Öne çıkan hizmetler

## 5. Personel / uzman / danışman bilgileri

- [ ] Ad soyad
- [ ] Görev/ünvan
- [ ] Kısa açıklama
- [ ] Fotoğraf
- [ ] WhatsApp / telefon

## 6. Appointment / Klinik / Veteriner özel alanları

- [ ] Hizmet tipleri
- [ ] Acil durum kabul ediliyor mu?
- [ ] Evcil hayvan türleri
- [ ] Çalışma saatleri
- [ ] Randevu saat aralığı

## 7. Salon özel alanları

- [ ] Hizmet fiyat listesi
- [ ] Kampanyalar
- [ ] Personel tercih edilebilir mi?
- [ ] Instagram galeri linki
- [ ] Öncesi/sonrası görselleri

## 8. Emlak özel alanları

- [ ] İlk ilan listesi
- [ ] Satılık/kiralık kategorileri
- [ ] Danışmanlar
- [ ] İlan fotoğrafları
- [ ] İlan açıklamaları
- [ ] Vitrin ilanları

## 9. Teknik bilgiler

- [ ] Firebase hesabı müşteri adına açılacak mı?
- [ ] Netlify/Vercel hesabı müşteri adına açılacak mı?
- [ ] Domain kimde?
- [ ] DNS erişimi var mı?
- [ ] Admin panel e-postası
- [ ] Teslim tarihi
